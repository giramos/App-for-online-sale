Historia

Historia

El Llagar Online nace como una propuesta para la evaluación extraordinaria de la asignatura Softwares y Estandares para la Web.

En esta página web se intenta mostrar los distintos conocimientos adquiridos durante las diferentes prácticas de la asignatura.

Para ello se aplicarán los conocimientos de HTML para realizar las diferentes vistas de la aplicación, de la misma manera se utilizarán los conocimientos adquiridos de CSS para mostrar de forma correcta y tratar los estilos de las vistas.

Por otro lado, también se utilizará XML para guardar las pizzas que conformarán la carta de la aplicación, junto con estos XML se crearán las DTDs y XMLSchemas necesarios para la validación, de la misma forma para mostrar las pizzas se utilizarán transformaciones XSLT.

En lo referente a la computación en el cliente se utilizará jquery para la creación de un reloj para las diferentes páginas de aplicación, el formato json será utilizado para el intercambio de información entre el cliente y el servidor y se utilizará Ajax para leer y mostrar la información en dicho formato.

También sera usado en lo que concierne a la computación en el cliente la api de geolocalización de HTML5, y se consumirán los servicios web de Google Maps para mostrar la ubicación de los locales a los clientes, asi como la ubicación del cliente respecto a estos, en caso de que autorice la geolocalización.

Por otro lado la computación en el servidor se hará sobre PHP, se utilizará para validar la información del cliente y guardarla en una base de datos MYSQL y se enviará desde PHP un mensaje a los usuarios de que se ha procesado su pedido correctamente.

El funcionamiento de todo lo anterior se probará sobre la máquina local XAMPP, y se utilizarán los validadores de HTML, CSS, XML, etc. para comprobar que están bien formados todos los ficheros que conforman el sitio web.
